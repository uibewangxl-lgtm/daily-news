# 每日新闻速览

每天 7:30 自动推送 20 条精选新闻到微信。

## 覆盖领域

| 领域 | 来源 |
|------|------|
| 时政·国际 | 联合早报、BBC中文、Reuters |
| 科技 | Hacker News、少数派、ArsTechnica |
| 财经·商业 | 36氪、TechCrunch、华尔街见闻 |
| 社会·民生 | 知乎日报、The Guardian |
| 科学·健康 | 果壳网、Nature |

## 本地测试

```bash
pip install -r requirements.txt
SENDKEY=你的SendKey python news_digest.py
```
